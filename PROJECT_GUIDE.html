Context API — Carrito

Métodos disponibles:

Método Descripción
addToCart(product) Agrega producto al carrito
removeFromCart(id) Elimina producto
updateQuantity(id, qty) Actualiza cantidad
getTotal() Retorna total en COP
getItemCount() Retorna cantidad de items
clearCart() Limpia todo el carrito

Uso en componentes:

import { useCart } from './context/CartContext';

export const MiComponente = () => {
  const { addToCart, cartItems, getTotal } = useCart();

  return (
    <>
      Total: ${getTotal()}
    </>
  );
};

Breakpoints Responsive

  • Mobile: Default (390px — tamaño iPhone)
  • Tablet: @media (min-width: 768px)
  • Desktop: @media (min-width: 1024px)

Nota: El proyecto es mobile-first — empieza en móvil y escala hacia arriba.

  • Tipografía: Luckiest Guy (display), Poppins (headers), Inter (body) — cargadas en index.html
  • Persistencia: Carrito en memoria — se borra al refrescar
  • Imágenes: Placeholders de Unsplash — reemplazar con URLs reales
  • WhatsApp: URL es placeholder — cambiar número real en Cart.jsx
